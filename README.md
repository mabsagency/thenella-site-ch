# THENELLA - Gospel Artist Official Website

Site officiel de THENELLA, artiste gospel. Explorez sa musique, ses vidéos et son parcours musical.

## Contenu

- **index.html** - Page principale du site
- **images/** - Dossier contenant toutes les images du site
- **GUIDE_IMAGES.md** - Guide pour ajouter les images

## Installation locale

1. Clonez ce repository
2. Ouvrez `index.html` dans votre navigateur

## Déploiement sur Vercel

### Option 1 : Via Git (Recommandé)

1. Poussez votre code sur GitHub, GitLab ou Bitbucket
2. Connectez-vous à [Vercel](https://vercel.com)
3. Cliquez sur "New Project"
4. Importez votre repository
5. Vercel détectera automatiquement qu'il s'agit d'un site statique
6. Cliquez sur "Deploy"

### Option 2 : Vercel CLI

```bash
npm install -g vercel
vercel
```
